# Documentación de Ventify

Bienvenido a la documentación oficial de Ventify, el sistema de punto de venta inteligente.

## Primeros Pasos

### 1. Iniciar Sesión
Ingresa a [app.ventify.com](https://app.ventify.com) con tu usuario y contraseña. Si eres nuevo, registra una cuenta gratuita.

### 2. Configurar tu Tienda
Ve a **Configuración** > **Negocio** para subir tu logo, definir tu dirección y configurar tus impuestos (IVA).

### 3. Agregar Productos
Dirígete a **Productos**. Puedes agregarlos uno por uno o importar un archivo CSV. Recuerda asignar precios y costos para ver tus ganancias.

---

## Facturación Electrónica (CFDI 4.0)

Ventify simplifica la emisión de facturas.
1. Realiza una venta.
2. Ve a **Reportes** > **Ventas**.
3. Selecciona la venta y haz clic en **Facturar**.
4. Ingresa el RFC del cliente y ¡listo!

> **Nota:** Necesitas configurar tu RFC y CSD en la sección de Configuración Fiscal antes de emitir tu primera factura.

---

## Soporte Offline

Ventify es una PWA (Progressive Web App). Esto significa que puedes instalarla en tu computadora o celular y usarla **sin internet**.
- Las ventas se guardan localmente.
- Cuando recuperas la conexión, el sistema sincroniza todo automáticamente.
